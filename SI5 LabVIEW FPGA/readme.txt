To open edit and compile this project, the following software is needed:

 - LabVIEW Full/Professional Development System 2014 (or newer)
 - LabVIEW FPGA Module 2014 (or newer)
 - NI RIO Device Drivers August 2014 (or newer)
 - NI FlexRIO Adapter Module Support 14.1 (or newer)
 - NI LabVIEW FPGA Xilinx ISE 14.7 Tools (or newer)
 - NI LabVIEW FPGA Xilinx Vivado 2013.4 Tools (or newer)
 
 To compile the FPGA bitfile a regeneration of the Xilinx Coregen support
 files might be neccessary. In LabVIEW, select
 Tools > FPGA Module > Regenerate IP Integration Node Support Files...
 In the following dialog open 'SI5 LabVIEW FPGA\FPGA Code\SI5 FPGA.vi'
 and select Select 'Regenerate Files...'
